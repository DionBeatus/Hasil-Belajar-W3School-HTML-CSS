Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web.

HTML memungkinkan pembuatan halaman web dengan struktur dan konten yang terorganisir menggunakan elemen-elemen seperti paragraf, tautan, gambar, dan lain-lain.

HTML Editor

Dokumen HTML dapat dibuat dengan editor teks sederhana atau editor HTML yang lebih canggih.

Editor sederhana seperti Notepad bisa dipakai, tapi editor khusus seperti Visual Studio Code menawarkan fitur yang memudahkan coding.

HTML Dasar

Struktur dasar HTML terdiri dari elemen-elemen utama seperti doctype, html, head, dan body.
